The following colors are assigned to each node according to its type

Type of node Color
Relation Brown
View Dark Green
Query Dark Blue
Attribute Light Gray
Condition Light Blue
Group By Light Green
Operand, Constant, Function White

The following colors are assigned to each node according to its state

State of node Color
If node has policies Yellow
If node has events Pink
If node is picked Cyan

Status of node Color
TO DELETE Red
TO DELETE CHILD Dark Brown
TO CHANGE DEPENDENT Purple
TO ADD CHILD Green
TO RENAME Orange
TO MODIFY Orange
EVENT BLOCKED Black
EVENT PROMPT Dark Gray
